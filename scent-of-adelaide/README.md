# The Scent of Adelaide | Premium Perfume Store

Welcome to the **Scent of Adelaide**, your premier boutique fragrance destination. This project is a high-fidelity, professional-grade e-commerce storefront built with a focus on luxury aesthetics and Australian compliance.

## 🚀 Store Features
- **Curated Selection**: 12 Men's and 18 Women's top-trending global perfume products.
- **AI Support Chatbot**: Integrated floating AI assistant for shipping, order tracking, and fragrance advice.
- **Australian Compliant**:
    - **ABN**: 82 697 448 668
    - **No GST** configuration for AUD transactions.
    - Full legal pages: Privacy, Refund Policy, Shipping Policy, and Disclaimer.
- **Modern UI**: Light pink, white, and black design language with gold accents.
- **Founder Showcase**: Featuring the full founding team (Darren Smith, Lee Bailie, et al.).
- **Newsletter Opt-in**: Automated 10% discount code generation via our "Inner Circle" loyalty program.
- **Product Tabs**: Seamless switching between Men's and Women's collections.
- **Payment & Logistics**:
    - Integrated Stripe gateway placeholders.
    - Automated CJ Dropshipping workflow logic.

## 🛠 Tech Stack
- **Frontend**: HTML5 Semantic Structure, CSS3 (Custom Design System with Variables).
- **Interactions**: Vanilla Javascript (Tabbed navigation, Modals, Chatbot logic).
- **Assets**: Custom-designed AI branding (Logo & Hero images).

## 📂 Project Structure
- `index.html`: Main store structure and SEO setup.
- `style.css`: Visual identity and responsive design system.
- `main.js`: Product data, founder injection, and interaction logic.
- `assets/`: Generated brand logo and hero visuals.

## 🏁 How to Run
To view the store live:
1. Open this directory in your favorite editor (e.g., VS Code).
2. Use **Live Server** or simply open `index.html` in your browser.
3. For the full "Pro" experience, ensure animations and transitions are loaded.
